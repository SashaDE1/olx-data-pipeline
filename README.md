# OLX Data Pipeline

Асинхронний пайплайн для веб-скрапінгу та обробки даних, написаний на Python. Проєкт збирає дані про комп'ютерні комплектуючі з OLX Україна, очищає їх за допомогою Pandas і виконує інкрементальне завантаження в Google Sheets для подальшої аналітики.

## 🚀 Головні можливості
* **Асинхронний збір:** Використовує `Playwright` та `asyncio` для швидкого, неблокуючого збору даних з різних міст України.
* **Відмовостійкість:** Вбудована обробка помилок (`try/except`), яка дозволяє ігнорувати "биті" посилання, таймаути браузера або тимчасові блокування, не зупиняючи весь процес парсингу.
* **Трансформація даних:** Застосовує `Pandas` для очищення сирого тексту, видалення дублікатів та розділення єдиного рядка на окремі стовпці "Місто" та "Дата".
* **Інкрементальне завантаження:** Підключається до Google Sheets через `gspread` та Google Service Account. Скрипт отримує вже існуючі в базі записи, порівнює їх і додає **лише нові, унікальні рядки** на основі URL-адреси оголошення. Це запобігає дублюванню даних при повторних запусках.

## 🛠️ Стек технологій
* **Мова:** Python 3.11+
* **Екстракція:** Playwright, asyncio
* **Трансформація:** Pandas
* **Завантаження:** Google Sheets API, gspread
* **Безпека:** python-dotenv (для керування секретами)

## ⚙️ Як запустити локально
1. Клонувати репозиторій.
2. Створити та активувати віртуальне середовище (`python -m venv OLX`).
3. Встановити залежності: `pip install -r requirements.txt`.
4. Встановити браузери Playwright: `playwright install chromium`.
5. Додати файл `.env` з посиланням на Google Таблицю (`GOOGLE_SHEET_URL=...`).
6. Додати файл `credentials.json` від сервісного акаунту Google в корінь проєкту.
7. Запустити парсер: `python main.py`.
